## Instructions
If you want to use your own backend. Clone the repo to your device.

`git clone https://github.com/SarthakS97/LLMsForSheets_Backend.git`

`cd LLMsForSheets_Backend`

Install all necessary packages
`npm i`

Run the code 
`npm start`
